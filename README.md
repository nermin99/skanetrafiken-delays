# Skånetrafiken Delays

Did the train you were on get delayed, but you can't remember which one it was? Or maybe it's winter and you can't even remember the last time the trains were not delayed? Fear no more, because skanetrafiken-delays.se keeps track of all delayed trains across the Öresund strait for you, both in real time and historically, so you can look back at past delays, patterns, and just how optimistic the timetable really was.

## Developing

### Local development

Run the following to start the project locally

```shell
npm start
```

### Deployment

Run the following to build and deploy the site to github pages

```shell
npm run deploy
```
