# Auto-form-filler

Puppeteer script that automates filling out Skånetrafiken's compensation claim form so you can apply for refunds without doing the clicking yourself.

## Prerequisites

- Node.js v24+

## Local development

Install dependencies:

```shell
npm install
```

Edit the `INPUT` object at the top of [auto-form-filler.js](auto-form-filler.js) with your ticket details, then run:

```shell
npm start
```
